# Flight Plan 2026-2031 · India → Australia / Germany

A single-file, 5-year career migration roadmap: from AI EdTech operations in India to a skilled role in Australia (primary) or Germany (secondary). Built as one self-contained `index.html`, designed to be re-read every week for five years.

## What's inside

- **14 tabs:** Overview, Profile (SWOT + gap-year scripts), Education decision (the Master's question, answered), Countries & Visas (verified June 2026 figures), Skills (15 ranked + beginner/intermediate/advanced roadmaps), 20 portfolio projects, Resources, Certifications, German A1→B2 track, Jobs & Strategy, Money, Daily Plan, Action Plan (90-day / 12-month / 5-year), and an Execution Checklist.
- **2026-verified figures:** EU Blue Card €50,700 / €45,934.20, Chancenkarte €13,092 (€1,091/month), Australia CSIT AUD 79,499 from 1 Jul 2026, visa fees AUD 4,910 (189/190) and AUD 3,210 (482), Germany's abolished 3-year fast-track citizenship, the 2026 Australian points-test reform.
- **Persistent checklist:** 47 items saved in your browser's localStorage. Progress feeds the bar in the nav and survives closing the tab. Device-local only (no sync, no account, no tracking).
- **Zero build, zero dependencies:** plain HTML/CSS/JS. Google Fonts load if online and degrade to system fonts if not. Dark mode follows your OS. Print shows all sections.

## Host it on GitHub Pages

1. Create a repository (public or private with Pages enabled), e.g. `flight-plan`.
2. Upload `index.html` and this `README.md` to the repository root.
3. Go to **Settings → Pages → Build and deployment**: Source = *Deploy from a branch*, Branch = `main`, Folder = `/ (root)`. Save.
4. Your roadmap is live at `https://<username>.github.io/<repo>/` within a minute or two.

Updating: edit `index.html` directly on GitHub (or locally and push). Pages redeploys automatically on every commit.

## Notes

- Checklist progress lives in localStorage under `roadmap-checks-v1`; the last open tab under `roadmap-tab-v1`. Clearing site data resets both. The "Reset all progress" button on the Checklist tab does the same with a confirmation.
- Direct links to tabs work via hash, e.g. `.../#checklist` or `.../#money`.
- All visa thresholds and fees index annually. Figures were verified June 2026; re-verify on immi.homeaffairs.gov.au and make-it-in-germany.com before any payment or lodgement.
- This is a planning document, not legal advice. Use one paid session with a MARA-registered agent (Australia) or a German immigration adviser at the decision points marked in the plan.
